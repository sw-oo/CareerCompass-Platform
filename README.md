# CareerCompass-Platform

### 서비스 주제 : 대학생들이 취업/대학원 진학 결정 시 필요한 정보를 한곳에서 검색할 수 있는 플랫폼

1. 서비스 목적

대학생들이 취업/대학원 진학을 결정할 때 필요한 정보를 한곳에서 제공
선배들의 실제 경험을 공유받고, 맞춤 추천을 받을 수 있는 플랫폼
흩어진 진로 정보를 통합 검색하고 트렌드를 파악

---

2. 주요 기능

-- 핵심 기능

(1) 회원 관리

 - 회원가입/로그인 (이메일, 소셜)
 - 프로필 관리 (학교, 학과, 관심 분야, 희망 진로)
 - 역할 구분 (재학생/멘토/기업)

(2) 통합 검색

 - 기업 정보 검색 (채용공고, 연봉, 기술스택, 복지)
 - 직무 정보 검색 (필요 역량, 커리어 패스)
 - 대학원 연구실 검색 (연구분야, 교수정보, 입학요건)
 - 실시간 트렌드 (인기 검색어, 떠오르는 직무)

(3) 추천 시스템

 - AI 기반 맞춤 진로 추천
 - 부족한 역량 분석 및 학습 자료 추천
 - 커리어 로드맵 생성

-- 부가기능

(4) 알림 기능

 - 관심 기업 채용 공고 알림
 - 관심 분야 새 후기 알림
 - 키워드 구독 (기업/직무/기술)

(5) 경험 공유

 - 후기 작성 (취업 합격, 대학원 입학, 인턴 경험)
 - Q&A 게시판 (진로 고민, 기업별 질문)

(6) 통계 대시보드

 - 학과별 취업률 및 주요 기업
 - 직무별 평균 연봉 추이
 - 개인 진로 준비도 점수
